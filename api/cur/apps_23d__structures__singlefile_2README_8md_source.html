1 3-D Structure Import Program
2 =============================
3 
4 This program reads ADDA and DDSCAT-formatted shapes, and then stores them
5 in HDF5 files. It provides an example of how to create and manipulate
6 shapes.

11 ```
12 3d_structures_example --from shape.dat --to shape.hdf5
